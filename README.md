# VEXORIUM — Digital Agency SaaS Dashboard

A premium, futuristic, glassmorphism multi-page SaaS dashboard prototype.

## Pages
- Landing: `/index.html`
- Auth: `/login.html`, `/signup.html`, `/forgot-password.html`, `/otp-verification.html`
- App: `/dashboard.html`, `/clients.html`, `/billing.html`, `/team.html`, `/settings.html`

## Run locally
Serve the folder with any static server, for example:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
